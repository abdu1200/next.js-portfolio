This ecommerce store showcases the power of Next.js for building dynamic
applications and integrates Stripe for secure, seamless payment processing. With
a focus on efficient online shopping, this app demonstrates the use of Stripe’s
API for payment flows within a modern ecommerce platform.

## Features

- **Stripe Payments**: Users can securely make payments through the Stripe API,
  with support for various payment methods.
- **Product Management**: Display products in a structured catalog with detailed
  information.
- **Shopping Cart**: Add items to the cart, view totals, and proceed to
  checkout.
- **User Authentication**: Customers can sign up, log in, and manage orders.
- **Responsive Design**: Optimized layout to ensure smooth usability across
  devices.

## Technologies

- **Next.js**: A powerful React framework for server-rendered and static sites.
- **Stripe API**: A comprehensive payment processing platform for secure
  transactions.
- **Tailwind CSS**: A utility-first CSS framework for designing a responsive UI.
- **Vercel**: Hosting and deployment service optimized for Next.js applications.

## Getting Started

To start with this project, clone the repository and install the dependencies:

```bash
git clone https://github.com/abdu1200/next-js-stripe-integration
cd nextjs-ecommerce-store
npm install
```

Then, set up environment variables for Stripe by creating a .env.local file with
the following keys:

NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=your_publishable_key
STRIPE_SECRET_KEY=your_secret_key

Once set up, run the development server:

```bash
npm run dev
```

The development server will start at http://localhost:3000.

## Deployment

Deploy this app to Vercel for optimal performance. Make sure to set up your
Stripe API keys as environment variables in Vercel for production.
